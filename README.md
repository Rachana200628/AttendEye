# AttendEye

**AttendEye** is an AI-powered smart classroom monitoring system designed to automate student attendance and analyze classroom attentiveness in real time. By combining Computer Vision, Deep Learning, and modern web technologies, the system helps educators gain valuable insights into student engagement while simplifying attendance management.

The application uses a **YOLOv5** model to detect and classify student behavior from live video streams, enabling the identification of attentive and inattentive students. A responsive web dashboard allows administrators and faculty to securely manage users, monitor classroom activity, and view analytics through an intuitive interface.

## Key Features

* AI-powered student attentiveness detection using YOLOv5
* Real-time classroom video analysis
* Automated attendance monitoring
* Secure user authentication with JWT
* Role-based access for administrators and faculty
* Interactive dashboard for monitoring and analytics
* Responsive and user-friendly interface
* RESTful API integration between the frontend, backend, and AI service

## Project Objective

The primary objective of AttendEye is to develop an intelligent classroom management system that automates attendance tracking and provides real-time insights into student attentiveness. The project demonstrates the integration of Artificial Intelligence with full-stack web development to build a scalable, efficient, and practical solution for smart education environments.
